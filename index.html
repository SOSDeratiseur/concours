Mode Manuel
Saisissez les participants vous-même et cochez les critères. Aucune configuration technique requise.

Mode Automatique API
Connectez l'API Facebook pour importer automatiquement les commentaires et détecter les tags.
⚠️
Limitations API Facebook : Les tags (3 personnes) sont vérifiés automatiquement. Les likes de page et partages publics restent à cocher manuellement (restriction Facebook depuis 2018).
📖 Comment obtenir un Token d'accès Facebook ?
  1. Allez sur developers.facebook.com et créez une App (type : Business).
  2. Dans votre App → Outils → Explorateur d'API Graph.
  3. Sélectionnez votre Page dans le menu déroulant.
  4. Cochez les permissions : pages_read_engagement et pages_read_user_content
  5. Cliquez sur Générer un token d'accès et copiez-le ci-dessous.
Formats acceptés : /posts/ID · permalink.php?story_fbid=… · /photo?fbid=…
